# Doomed
É um jogo que eu criei em Unity usando C# e um pacote de tiles personalizaveis que se encontra na loja da Unity desenvolvido pela Unity.

O jogo está totalmente funcional e já é possível edita-lo sem programação apenas pela UI.

Porém eu quero adicionar uma detecçao de colisão no eixo que é ignorado pelo projeto 2d para deixa-lo 3d de fato. Isso seria executado com melhores práticas se eu refizesse o environment como 3d, e enquanto eu não fizer isso o jogo não será publicado.
